# Orchestrate Launch Campaign

Single `openai_chat` call (json_mode) with a system prompt that demands the
full structured payload below. The skill does no image work — it composes the
*plan* layer that sits on top of the visuals already generated by earlier
slices (social pack, packaging, website).

Output shape:

```json
{
  "calendar": [{ "day_offset": -14, "channel": "email",  "asset": "Teaser", ... }],
  "emails":   [{ "slot": "announce", "subject": "...", "preheader": "...", "body": "...", "cta": "..." }],
  "press_kit": {
    "one_liner": "...",
    "short_bio": "...",
    "long_bio":  "...",
    "quotes":    ["...", "...", "..."],
    "contact_placeholder": "press@<brand-domain>.com"
  },
  "ads": {
    "meta":   [{ "headline": "...", "primary_text": "...", "cta": "..." }],
    "google": [{ "headline": "...", "description": "...", "path": "..." }],
    "tiktok": [{ "hook": "...", "script_beats": ["..."], "cta": "..." }]
  },
  "rollout": {
    "phases": [
      { "name": "Tease",   "days": "T-14 to T-3", "focus": "..." },
      { "name": "Launch",  "days": "T-0 to T+2",  "focus": "..." },
      { "name": "Sustain", "days": "T+3 to T+14", "focus": "..." }
    ],
    "success_metrics": ["..."]
  }
}
```
